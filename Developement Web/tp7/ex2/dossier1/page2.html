Les Tableaux

Les tableaux en HTML sont utilisés pour organiser et afficher des données de manière structurée. Ils se créent avec la balise < table>, à l’intérieur de laquelle on définit des lignes (< tr> pour "table row") et des cellules. Chaque cellule peut être une cellule d'en-tête (< th> pour "table header") ou une cellule standard (< td> pour "table data"). Les en-têtes de colonnes aident à donner un titre aux données, tandis que les cellules contiennent les informations. Bien que les tableaux puissent servir pour organiser des mises en page, il est préférable de les utiliser uniquement pour présenter des données structurées, car CSS et flexbox sont plus adaptés pour la mise en forme visuelle des pages.
